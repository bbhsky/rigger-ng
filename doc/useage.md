#使用指南


## 系统控制

### 项目初始化

``` shell
cd <you-project>
rg init
```

### 编辑 run.yaml

### 指令
``` shell
cd <you-project>
    rg conf  -s test -e dev
    rg start
    rg stop
    rg restart
    rg reload
    rg clean
    rg help
    rg help res
```
